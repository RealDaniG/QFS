# Zero-Sim Tools

This directory contains Zero-Simulation compliance tools and utilities.

## Files

- See `../../scripts/zero_sim_analyzer.py` for the main analyzer
- See `../../scripts/zero_sim_autofix.py` for automated fixes
- See `../../scripts/zero_sim_dashboard.py` for compliance dashboard

## Usage

Run from repository root:

```bash
python v13/scripts/zero_sim_analyzer.py --enforce-strict --target v13/atlas/
```
